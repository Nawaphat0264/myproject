# แบบข้อเสนอโครงงาน

Dew & Name online food

## สมาชิก

1. ชื่อ นวพรรษ สายพิณ 6730300264
2. ชื่อ ภูรินทร์ ราญมีชัย 6730300493

## รายละเอียดโดยย่อ

โครงงานนี้เป็นการสร้างใบเมนูออนไลน์ (creating an online menu.) ซึ่งมีวัตถุประสงค์เพื่ออำนวยความสะดวกให้เเก่ลูกค้าโดยไม่ต้องรอใบเมนู สามารถดูเมนูในเเอฟได้เลยเเละสามารถสั่งอาหารได้เลย

## คุณลักษณะและขอบเขต

### คุณลักษณะขั้นต่ำ

- มีหน้าล็อกอินที่มีการใส่ชื่อจอง การจองโต๊ะ เมนูเเละเลือกเวลาที่จะมา
- มีหน้าเมนูในการเลือกอาหาร สามารถเพิ่มหรือลดรายการอาหารได้เเล้วสามารถคลิกไปที่คิดเงินได้
- มีหน้ารวมรายการที่สั่งเเละการชำระเงิน

### คุณลักษณะเพิ่มเติม

- เพิ่มช่วงโปรโมชั่น
- เพิ่มรายการอาหารให้มากขึ้น

## ความท้าทายและความเสี่ยง

1. **ความท้าทายด้านเทคนิค**:
    - ยังเขียนโค้ดภาษา java ไม่คล่อง
    - **แนวทางแก้ไข**: ศึกษาและเรียนรู้จากลิงก์ w3schools (https://www.w3schools.com/java/)

2. **ความท้าทายด้านการทำโปรเจค**
    - ยังไม่เคยลองทำโปรเจค อาจทำให้โปรเจคไม่เสร็จสมบูรณ์ก่อนส่ง
    - **แนวทางแก้ไข**: ศึกษาเเนวทางการทำโปรเจคจาก youtube ช่อง Bitter Code
      
2. **ความท้าทายในการใช้งาน javaFX**
    - ยังไม่คุ้นเคยกับ JavaFX ในการทำ GUI
    - **แนวทางแก้ไข**: ศึกษาเเละเรียนรู้จากลิ้งค์ (https://www.pragmaticcoding.ca/beginners/intro)

3. **ความเสี่ยงด้านเวลา**:
    - หากใช้เวลาในการศึกษาโค้ดภาษา java มากเกินไป อาจทำให่โปรเจคทำส่งไม่ทัน
    - **แนวทางแก้ไข**: วางแผนฝึกทักษะคู่ไปกับการทำโปรเจค และติดตามความคืบหน้า (progress) ของตนเองทุกสัปดาห์
